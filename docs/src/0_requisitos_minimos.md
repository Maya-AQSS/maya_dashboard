Notificaciones

al pulsar mas se enviara al usuario a la vista de tipo tabla de notificaciones
Aplicaciones favoritas al aside izquierdo
el boton mas te envia a gestion de aplicaciones para seleccionar favoritos
Dashboard tendra bienvenida y resumen diario
