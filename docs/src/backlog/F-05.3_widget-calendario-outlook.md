**Descripción:**
Como usuario del dashboard con cuenta Microsoft vinculada,
Quiero ver mis eventos del calendario Outlook en el widget del dashboard,
Para planificar mi jornada sin necesidad de abrir Outlook o Teams por separado.

**Criterios de Aceptación (Checklist):**

- [ ] **Escenario 1 (Widget omitido hasta disponibilidad):**
  **Dado que** la integración Microsoft Graph no está operativa en el MVP inicial,
  **Cuando** el grid de widgets se renderiza,
  **Entonces** el slot del widget de calendario NO se muestra en la UI hasta que esta feature esté disponible (el slot puede quedar vacío o mostrar estado "próximamente").

- [ ] **Escenario 2 (Vista diaria de eventos):**
  **Dado que** el usuario tiene cuenta Microsoft vinculada y la integración está activa,
  **Cuando** el widget de calendario carga,
  **Entonces** muestra los eventos del día actual del calendario Outlook del usuario con título, hora de inicio y duración.

- [ ] **Escenario 3 (Rango cambiable por el usuario):**
  **Dado que** el widget muestra vista diaria por defecto,
  **Cuando** el usuario selecciona un rango diferente (ej. esta semana),
  **Entonces** el widget actualiza los eventos mostrados según el rango seleccionado.

- [ ] **Escenario 4 (Sin cuenta Microsoft — estado de conexión):**
  **Dado que** el usuario no tiene cuenta Microsoft vinculada,
  **Cuando** el widget intenta cargar eventos,
  **Entonces** muestra el estado "Conecta tu cuenta Microsoft" con un enlace o botón para iniciar el proceso de vinculación (F-05.2).

- [ ] **Escenario 5 (Solo lectura):**
  **Dado que** el widget es de solo lectura,
  **Cuando** el usuario interactúa con los eventos,
  **Entonces** solo puede verlos. No existe ninguna opción para crear, editar ni eliminar eventos desde el dashboard.

- [ ] **Requisito Técnico (Auditoría):**
  El evento "conectar cuenta Microsoft" debe registrarse en el audit log (F-08.1) con el tipo `CALENDAR`.

**Notas:**

- El widget está completamente omitido en la UI hasta que la integración esté operativa. No se muestra ni en estado vacío ni en error durante el MVP.
- Vista diaria por defecto, cambiable por el usuario al rango que necesite.
- Solo lectura: no permite crear ni modificar eventos de Outlook.
- Este widget es el tercero de los 3 widgets fijos del MVP en el grid, pero su activación es `Could`.
